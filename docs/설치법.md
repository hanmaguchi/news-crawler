# 설치 가이드

## 사전 준비

### 1. Python 3.12 이상 확인

```bash
python --version   # Python 3.12.x 이상이어야 함
```

Python이 없거나 버전이 낮으면 [python.org](https://www.python.org/downloads/) 에서 설치한다.

### 2. uv 설치

`uv`는 빠른 Python 패키지 관리자다. 없으면 아래 명령으로 설치한다.

```bash
# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows (PowerShell)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

설치 후 터미널을 재시작하거나 `source ~/.bashrc`(또는 `~/.zshrc`)를 실행한다.

```bash
uv --version   # uv 0.x.x 확인
```

### 3. Git 확인

```bash
git --version
```

## 의존성 설치

```bash
uv sync
```

`uv sync`는 `pyproject.toml`에 명시된 모든 패키지를 `.venv`에 설치한다.
별도의 `pip install` 없이 이 한 줄로 완료된다.

## 네이버 API 키 발급 (선택 — 구글만 쓸 경우 건너뜀)

> 키 없이도 구글 News RSS만으로 바로 사용할 수 있다.
> 네이버 출처까지 활성화하려면 아래 절차를 따른다.

1. [네이버 개발자 센터](https://developers.naver.com/apps) 접속 후 로그인
2. **Application → 애플리케이션 등록** 클릭
3. 애플리케이션 이름 입력 (예: `뉴스크롤러`)
4. **사용 API** 목록에서 **검색** 체크 후 저장
5. 등록 후 **Client ID**와 **Client Secret** 복사

### 키 설정

프로젝트 루트에 `.env` 파일을 생성한다 (`.env.example`을 복사해서 수정):

```bash
cp .env.example .env
```

`.env` 파일을 열고 값을 채운다:

```dotenv
NAVER_CLIENT_ID=발급받은_Client_ID
NAVER_CLIENT_SECRET=발급받은_Client_Secret
```

> ⚠️ `.env` 파일은 `.gitignore`에 포함되어 있어 Git에 올라가지 않는다.
> 절대 GitHub에 직접 커밋하지 않는다.

## 실행 확인

```bash
uv run streamlit run app.py
```

브라우저에서 `http://localhost:8510` 이 자동으로 열리면 설치 완료다.

---

## 포트 변경 (선택)

기본 포트는 `8510`이다. 변경하려면 `.streamlit/config.toml`을 수정한다:

```toml
[server]
port = 8510   # 원하는 포트 번호로 변경
```

## 문제 해결

### `uv: command not found`
`uv` 설치 후 터미널을 재시작하거나 PATH를 갱신한다.

```bash
source ~/.bashrc   # 또는 source ~/.zshrc
```

### `ModuleNotFoundError`
`uv sync`를 재실행한다. `.venv`가 없거나 패키지가 불완전하게 설치된 경우다.

### 네이버 `401 Unauthorized`
`.env`의 `NAVER_CLIENT_ID` / `NAVER_CLIENT_SECRET` 값이 정확한지 확인한다.
앱 설정에서 **검색 API**가 활성화되어 있는지도 확인한다.

### 포트 충돌 `Address already in use`
다른 포트를 지정해 실행한다:

```bash
uv run streamlit run app.py --server.port 8511
```
